# CHM 转文件夹 —— 流程归档

本文件夹包含将华为交换机 CHM 命令参考转换为层级文件夹的完整流程记录。

## 文件清单

| 文件 | 说明 |
|------|------|
| `流程文档.md` | 完整的五步流程说明，包含技术细节、遇到的问题和解决方案 |
| `关键Prompt记录.md` | 三个关键用户指令，它们决定了最终产物的形态 |
| `命令参考.md` | 可复用的 shell 命令集合 |
| `chm_to_folders.py` | **核心脚本**：解析 .hhc → 提取纯文本 → 创建层级文件夹 |
| `extract_command_funcs.py` | 辅助脚本：提取所有命令的"命令功能"段 |
| `generate_depth3_readmes.py` | 辅助脚本：按关键词匹配为 177 个命令类生成 README |
| `代码与README生成说明.md` | 代码设计说明和 README 生成策略 |

## 核心思路

```
CHM → 7z解包 → 解析.hhc目录树 → 提取HTML纯文本 → 重建文件夹层级
                                                    ↓
                         README.md导航体系 ← 读取命令功能段 ← 提取
```

## 关键技术点

1. **编码**：.hhc 和 HTML 都是 GB2312，需正确处理
2. **自闭合标签**：`<meta>` 等标签导致 Python HTMLParser skip_depth 计数错误
3. **长路径**：Windows MAX_PATH 限制，使用 `\\?\` 前缀解决
4. **层级判定**：通过 `<UL>` 嵌套数确定目录深度
