# CHM 转文件夹 —— 完整命令参考

## 环境准备

```bash
# 安装 7-Zip（Windows）
winget install 7zip.7zip --accept-package-agreements --accept-source-agreements
```

## 第一步：解包 CHM

```bash
mkdir -p chm_extracted

"/c/Program Files/7-Zip/7z.exe" x \
  "S1700, S5700, S6700 V600R025C00 命令参考.chm" \
  -o"chm_extracted" \
  -y
```

解包后检查：
```bash
ls chm_extracted/
# 应看到：#STRINGS, #SYSTEM, #TOPICS, images/, v6r25c00/, toctopics/, *.hhc, *.hhk
ls chm_extracted/*.hhc
# zh-cn_bookmap_0000001569270929.hhc
ls chm_extracted/v6r25c00/CLI/ | head
# dc_User_Command_Reference_title.html, ALIASCONFIG(CLIOM).html, ...
```

## 第二步：运行转换脚本

```bash
python chm_to_folders.py
```

输出：
- `命令参考_文件夹/` —— 带层级结构的纯文本文件夹

## 第三步：生成底层 README

```bash
# 提取命令功能描述
python extract_command_funcs.py

# 生成 177 个 depth-3 README.md
python generate_depth3_readmes.py
```

## 第四步：人工编写中层和顶层 README

中层 README（20 个章节级）和顶层 README（2 个）需要人工根据子目录内容总结编写，参见流程文档中的说明。

## 最终产物结构

```
命令参考_文件夹/
├── README.md
├── 前言.txt
├── 不支持命令列表.txt
└── 命令参考/

    ├── 基础配置/
    │   ├── README.md
    │   ├── 首次登录设备配置命令/README.md + *.txt
    │   └── ...
    ├── IP路由/README.md + ...
    └── ... (20 chapters, 177 command groups)
```
