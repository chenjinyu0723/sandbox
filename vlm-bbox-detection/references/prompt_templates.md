# Agent 迭代标注 - 输出模板

## 每轮迭代必须输出的格式

Agent 看图后，**必须逐物体说出位置和调整**：

```
### 第 {N} 轮

**扫描枪1**：位于图片左下角蓝色工作台上，当前 bbox [120, 200, 320, 380]
→ 右边界偏左，没包住枪头 → 向右扩展 20px → 新 bbox [120, 200, 340, 380]

**扫描枪2**：位于图片中部偏右的抽屉旁，当前 bbox [600, 180, 750, 350]
→ 框准确，不动

**护目镜1**：位于右上角桌面边缘，当前 bbox [510, 160, 690, 290]
→ 上边界偏高，没包住镜框上沿 → 向上扩展 15px → 新 bbox [510, 145, 690, 290]

---
本轮调整了 2 个框，仍需继续 / ✅ 全部准确
```

## 第一轮（初始标注）

```
### 第 1 轮 — 初始标注

**扫描枪1**：位于图片左下角工作台上，bbox [120, 200, 350, 400]
**扫描枪2**：位于中部偏右抽屉旁，bbox [600, 180, 780, 360]
**护目镜1**：位于右上角桌面边缘，bbox [510, 160, 690, 290]

---
共标注 3 个物体，进入验证轮
```

## 调整方向术语

| 问题 | 方向 | 示例 |
|------|------|------|
| 左边没包住 | 向左扩展 | x1 -= 15px |
| 右边没包住 | 向右扩展 | x2 += 15px |
| 上边没包住 | 向上扩展 | y1 -= 10px |
| 下边没包住 | 向下扩展 | y2 += 10px |
| 框太大 | 缩小/收紧 | 各方向缩 5-10px |
| 框整体偏左 | 向右移动 | x1, x2 都 +20px |
| 框整体偏右 | 向左移动 | x1, x2 都 -20px |
| 完美 | 准确，不动 | — |
